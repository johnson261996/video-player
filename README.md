# video_player

This is a simple video player which user can play videos and can play the videos from pre defined templates.


## Directory Structure

```
lib
│
└───models
 |    |──video.dart  
 └───screens
 |    |──landscape_screen.dart
 |    |──video_player_from_file.dart
 |   
 |    
 └───util
 |     |──size_config.dart  
 │───main.dart
```
